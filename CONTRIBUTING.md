# Katkıda Bulunma Rehberi

Projeye katkıda bulunmak için teşekkürler! 🎉

## Geliştirme Ortamı Kurulumu

1. Android Studio Hedgehog veya sonrasını indirin
2. JDK 17 veya sonrasını kurun
3. Repoyu klonlayın

## Kod Standartları

- Kotlin Coding Conventions'a uygun yazın
- Compose fonksiyonları büyük harfle başlamalı (PascalCase)
- Değişkenler camelCase olmalı
- Sabitler UPPER_SNAKE_CASE olmalı

## Commit Mesajları

Conventional Commits formatını kullanın:

- `feat:` Yeni özellik
- `fix:` Hata düzeltmesi
- `docs:` Dokümantasyon değişiklikleri
- `style:` Kod formatı değişiklikleri
- `refactor:` Yeniden yapılandırma
- `test:` Test eklemeleri/düzeltmeleri
- `chore:` Derleme/deployment değişiklikleri

## Pull Request Süreci

1. Branch'inizi `main`den güncel tutun
2. Tüm testlerin geçtiğinden emin olun
3. Kod review talep edin
4. CI/CD pipeline'ının başarılı olduğundan emin olun
